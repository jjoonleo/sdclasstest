# sdclasstest

Static school utility pages for class links and timetable access.

## Why This Project Matters

This is a small early frontend project that collects useful school links and
class meeting room links in simple HTML/CSS pages.

## Contents

- Student class page
- Teacher meeting-room page
- Grade/class link page
- CSS styling
- Timetable image

## Local Preview

Open `sdclass.html` or `sdclass_teacher.html` in a browser.

## Portfolio Highlight

This should be treated as an archive project. It shows early practical frontend
work but is not a primary portfolio candidate.
